# 1000 НЕТ

Интерактивный трекер отказов для сетевиков.

Главная идея сервиса: не считать продажи, регистрации и «да». Считать только действия и устойчивость к отказам.

## Что уже есть в MVP

- Главный дашборд с уровнем, прогрессом `НЕТ/1000`, днём пути и фразой дня.
- Интерактивная таблица из 1000 ячеек: 40 колонок × 25 рядов.
- Кнопка `+1 НЕТ`.
- Кнопка отмены последнего действия.
- 7 уровней прогресса:
  - 10 НЕТ — Первые шаги
  - 50 НЕТ — Уже не страшно
  - 100 НЕТ — Вышел из ступора
  - 250 НЕТ — Стабильный игрок
  - 500 НЕТ — Железная психика
  - 750 НЕТ — Машина действий
  - 1000 НЕТ — Легенда отказов
- 12 цветовых тем.
- Светлая и тёмная версия.
- JSON-библиотека мотивации: 50 начал × 20 окончаний = 1000 уникальных фраз.
- Сохранение прогресса в `localStorage`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Основная логика

Файл с базой мотивационных фраз: `src/data/quotes.json`.

Генерация 1000 фраз, уровни, темы и логика кликов: `src/App.tsx`.

Стили интерфейса: `src/styles.css`.
